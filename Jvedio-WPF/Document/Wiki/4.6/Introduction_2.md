# 运行环境


>运行环境：.NET FrameWork 4.7 [点此下载](https://dotnet.microsoft.com/download/dotnet-framework)


## 系统支持

>本软件在 Win10 X64 上通过测试，在虚拟机的 Win7 X64 和 Win8 X64 上测试通过


| 操作系统 | 是否支持 |
| :------: | :------: |
|  Win XP  |    ×     |
|  Win 7   |    √     |
|  Win 8   |    √     |
|  Win 10  |    √     |


## 最低配置

| 硬件配置 |         参数         |
| :------: | :------------------: |
|  处理器  | 1千兆赫 (GHz) 或更快 |
|   内存   |       2GB 以上       |
|   硬盘   |      16GB 以上       |
|   显卡   |   DirectX 9  以上    |
